初始化权重

现在我们考虑训练之中对于w超参数的取值当然不能简单地取值（比如取全零或者全等，则使得每一层神经网络的效果等价，失去训练的价值）这也许会导致正向传播时的信号不稳定，或者反向传播梯度爆炸或消失的问题。

一个自然的想法是，我们考虑使得输出的下一层于上一层的神经元分布一致，这样，我们便能够保证整体在传播过程之中信号的稳定性。

1.Xavier初始化

Xavier初始化针对这样一类激活函数：它们在零的附近近似线性，且具有零两侧的中心对称性。（比如说sigmoid和tanh函数）

这种初始化方法还有一个关键假设，即每一层的权重、输入都是满足均值为零的独立同分布，下面我们希望通过推导，找出w应该满足的方差，使得输出层与其输入层的分布一致。

先考虑输出层z方差的表达式：

Var(zil)=Var(∑j=1nl−1Wijlajl−1)(1.1)\begin{array}{cc} \text{Var}(z_i^l) = \text{Var}\left(\sum_{j=1}^{n_{l-1}} W_{ij}^l a_j^{l-1}\right)\tag{1.1} \end{array} Var(zil​)=Var(∑j=1nl−1​​Wijl​ajl−1​)​(1.1)

又由于变量w,a彼此独立，且其均值为0，那么由独立变量的方差公式有：

Var(zil)=∑j=1nl−1Var(Wijlajl−1)=∑j=1nl−1Var(Wijl)Var(ajl−1)(1.2)\begin{array}{cc} \text{Var}(z_i^l) = \sum_{j=1}^{n_{l-1}} \text{Var}(W_{ij}^l a_j^{l-1}) = \sum_{j=1}^{n_{l-1}} \text{Var}(W_{ij}^l) \text{Var}(a_j^{l-1})\tag{1.2} \end{array} Var(zil​)=∑j=1nl−1​​Var(Wijl​ajl−1​)=∑j=1nl−1​​Var(Wijl​)Var(ajl−1​)​(1.2)

再由于同分布，所以有：

Var(zil)=nl−1σw2σa2(l−1)(1.3)\begin{array}{cc} \text{Var}(z_i^l) = n_{l-1} \sigma_w^2 \sigma_a^{2(l-1)}\tag{1.3} \end{array} Var(zil​)=nl−1​σw2​σa2(l−1)​​(1.3)

而由于在0附近这类函数接近线性，因此激活函数不改变方差的值。

所以，对于正向传播，我们便获得了使得输入与输出地方差相等的权重方差：

σw2=1nl−1\begin{array}{cc} \boxed{\sigma_w^2 = \frac{1}{n_{l-1}}} \end{array} σw2​=nl−1​1​​​

事实上，我们还需要反向传播也满足这样的情况，于是对于反向传播，同理有：

σw2=1nl\begin{array}{cc} \boxed{\sigma_w^2 = \frac{1}{n_l }} \end{array} σw2​=nl​1​​​

两者取调和平均，获得了第l层所要求的w方差：

σw2=2nl−1+nl\begin{array}{cc} \sigma_w^2 = \frac{2}{n_{l-1} + n_l} \end{array} σw2​=nl−1​+nl​2​​

这是一个使得向前和反向传播信号均稳定的折中方案。

2.HE初始化

现在我们考虑另一种非对称的激活函数，ReLU函数。依照Xavier的想法，我们来推导其对应的方差。

值得一提，Xavier初始化在ReLU这类激活函数中失效的原因是在经过其激活后，无法保证前后均值为零，从而使得以上推导无法进行。

下面我们来推导Leaky ReLU激活函数的情形：（可以把经典的ReLU当成是Y轴左侧斜率为零的特例。）

这里我们依然假设每一层的权重、输入都是满足均值为零的独立同分布，并且为对称分布。

写出Leaky ReLU函数：

ϕ(z)={z,z≥0az,z<0\begin{array}{cc} \phi(z) = \begin{cases} z, & z \ge 0 \\ a z, & z < 0 \end{cases} \end{array} ϕ(z)={z,az,​z≥0z<0​​

下面我面仍然分析输出层与输入层之间的方差关系：

由于独立同分布，式(1.1)至式（1.3）的推导仍然适用，现在我们关心激活函数将如何影响方差。

z的分布关于 0 对称（零均值高斯初始化时近似成立），设其概率密度函数为p(z),则显然，p(z)是偶函数。

以下推导暂时忽略激活后均值非零的影响，因为实际上的影响并不大！！！

Var(y)=E[y2]=∫0∞z2p(z)dz+a2∫−∞0z2p(z)dz\begin{array}{cc} Var(y)=\mathrm{E}[y^2] = \int_{0}^{\infty} z^2 p(z) dz + a^2 \int_{-\infty}^{0} z^2 p(z) dz \end{array} Var(y)=E[y2]=∫0∞​z2p(z)dz+a2∫−∞0​z2p(z)dz​

又由于分布是对称的，我们有：

∫−∞0z2p(z)dz=∫0∞z2p(z)dz=12E[z2]\begin{array}{cc} \int_{-\infty}^{0} z^2 p(z) dz = \int_{0}^{\infty} z^2 p(z) dz = \frac12 \mathrm{E}[z^2] \end{array} ∫−∞0​z2p(z)dz=∫0∞​z2p(z)dz=21​E[z2]​

综合以上两式，我们可以得到：

E[y2]=1+a22σz2\begin{array}{cc} \mathrm{E}[y^2] = \frac{1+a^2}{2} \sigma_z^2 \end{array} E[y2]=21+a2​σz2​​

再由前（1.3）式，我们获得权重方差：

σw2=2(1+a2)⋅nl\begin{array}{cc} \boxed{\sigma^2_{w} = \frac{2}{(1+a^2) \cdot n_l}} \end{array} σw2​=(1+a2)⋅nl​2​​​

当a=0时，此方差即为ReLU对应的HE初始化。

3.评述

这一类初始化的核心假设都是权重以及输入层服从独立同分布。因此，最为关键的是找到激活函数将如何影响方差。（比如tanh、sigmoid的不变；ReLU的半衰减）

*后记

关于Leaky ReLU中激活后均值对方差影响的推导：

考虑经过激活后的均值：

E[y]=E[ϕ(z)]=∫−∞∞ϕ(z)p(z)dz\begin{array}{cc} \mathrm{E}[y] = \mathrm{E}[\phi(z)] = \int_{-\infty}^{\infty} \phi(z) p(z) dz \end{array} E[y]=E[ϕ(z)]=∫−∞∞​ϕ(z)p(z)dz​

E[y]=∫0∞z p(z)dz+a∫−∞0z p(z)dz\begin{array}{cc} \mathrm{E}[y] = \int_{0}^{\infty} z\, p(z) dz + a \int_{-\infty}^{0} z\, p(z) dz \end{array} E[y]=∫0∞​zp(z)dz+a∫−∞0​zp(z)dz​

由于 p(z) 对称：

∫−∞0z p(z)dz=−∫0∞z p(z)dz\begin{array}{cc} \int_{-\infty}^{0} z\, p(z) dz = -\int_{0}^{\infty} z\, p(z) dz \quad \end{array} ∫−∞0​zp(z)dz=−∫0∞​zp(z)dz​

令：

μ+=∫0∞z p(z)dz>0(*)\begin{array}{cc} \mu_+ = \int_{0}^{\infty} z\, p(z) dz > 0\tag{*} \end{array} μ+​=∫0∞​zp(z)dz>0​(*)

则有：

E[y]=(1−a)μ+\begin{array}{cc} \mathrm{E}[y]=(1-a)\mu_+ \end{array} E[y]=(1−a)μ+​​

由上式可以看出，实际上经过激活后的均值并不为零，理论上应该予以考虑：

σy2=E[y2]−(E[y])2\begin{array}{cc} \sigma^2_{y} = \mathrm{E}[y^2] - (\mathrm{E}[y])^2 \end{array} σy2​=E[y2]−(E[y])2​

下面从正态分布中计算式（*）的积分：

μ+=∫0∞z2πσze−z22σz2dz=σz2π\begin{array}{cc} \mu_+ = \int_{0}^{\infty} \frac{z}{\sqrt{2\pi}\sigma_z} e^{-\frac{z^2}{2\sigma_z^2}} dz = \frac{\sigma_z}{\sqrt{2\pi}} \end{array} μ+​=∫0∞​2π​σz​z​e−2σz2​z2​dz=2π​σz​​​

联立以上所有式子，得到：

σy2=[1+a22−(1−a)22π]σz2\begin{array}{cc} \sigma_y^2 = \left[ \frac{1+a^2}{2} - \frac{(1-a)^2}{2\pi} \right] \sigma_z^2 \end{array} σy2​=[21+a2​−2π(1−a)2​]σz2​​

这是最为严格的表达式，但在实际常用的参数 a 很小（如 0.01）且 z方差较小时，这个均值相对较小。在 HE初始化的原始推导中，为了简化，常假设激活后均值为 0，而严格推导时会发现这个均值非零，但影响不大。
