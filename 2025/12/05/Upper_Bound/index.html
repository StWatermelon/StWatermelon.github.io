泛化误差上界的证明

发表于2025-12-05|更新于2025-12-10|学习笔记
|浏览量:

现在我们希望给出假设空间模型的泛化误差上界的证明，这个证明相对比较繁琐，以下略去对于VC维的讨论，并将它简单地看作是假设空间的容量（对于VC维的讨论，也许以后会有。。。）

PART1.证明前的准备工作

1.Markov定理的证明：

设 X 是一个非负的随机变量，则对于任意正实数a,有：

P(X≥a)≤E[X]aP(X \ge a) \le \frac{\mathbb{E}[X]}{a} P(X≥a)≤aE[X]​

定义指示函数：

I{X≥a}={1if X≥a,0if X<a.I_{\{X \ge a\}} = \begin{cases} 1 & \text{if } X \ge a,\\ 0 & \text{if } X < a. \end{cases} I{X≥a}​={10​if X≥a,if X<a.​

我们容易发现，**I(x)的平均值就是X不小于a的概率！**于是我们致力于证明这个函数的上界：

因为X≥0，且当X≥a时显然有X≥a⋅I{X≥a}；而当X<a时，I{X≥a}=0，此时X≥0=a⋅0\begin{array}{cc} 因为 X \ge 0，且当 X \ge a 时显然有 X \ge a \cdot I_{\{X \ge a\}}；\\而当 X < a 时，I_{\{X \ge a\}} = 0，此时 X \ge 0 = a \cdot 0 \end{array} 因为X≥0，且当X≥a时显然有X≥a⋅I{X≥a}​；而当X<a时，I{X≥a}​=0，此时X≥0=a⋅0​

因此，不论X如何取值，我们总有：

Xa≥I{X≥a}\frac{X}{a} \ge I_{\{X \ge a\}} aX​≥I{X≥a}​

对上式取平均。即可证明Markov定理。

2.Horffding不等式的证明：

先证明下面引理:

对所有满足E[X]=0且a≤X≤b(a<0<b)的分布E[eλX]≤exp⁡(λ2(b−a)28)\begin{array}{cc} 对所有满足\mathbb{E}[X]=0 且 a \leq X \leq b (a<0<b)的分布\\ \mathbb{E}[e^{\lambda X}] \le \exp\left( \frac{\lambda^2 (b-a)^2}{8} \right) \end{array} 对所有满足E[X]=0且a≤X≤b(a<0<b)的分布E[eλX]≤exp(8λ2(b−a)2​)​

原先我的计划是通过Taylor展开到二阶来考虑这个结果，但是e指数Taylor展开并不能获得上界的放缩（可以获得下界的放缩，因为其余项恒正）于是我们必须考虑新的放缩方式。注意到e指数函数的凸性，下面考虑Jensen不等式放缩：（这样可以出现上界）

凸函数的琴声不等式是：

f((1−θ)a+θb)≤(1−θ)f(a)+θf(b)\begin{array}{cc} f((1-\theta)a + \theta b) \leq (1-\theta)f(a) + \theta f(b) \end{array} f((1−θ)a+θb)≤(1−θ)f(a)+θf(b)​

我们需要的是X，因此，将X写成端点值的组合：

令θ=X−ab−a，则θ∈[0,1]，且X=(1−θ)a+θb\begin{array}{cc} 令 \theta = \frac{X - a}{b - a} ，则 \theta \in [0, 1] ，且 X = (1-\theta)a + \theta b \end{array} 令θ=b−aX−a​，则θ∈[0,1]，且X=(1−θ)a+θb​

我们将e指数函数代入：

eλX≤(1−θ)eλa+θeλb\begin{array}{cc} e^{\lambda X} \leq (1-\theta) e^{\lambda a} + \theta e^{\lambda b} \end{array} eλX≤(1−θ)eλa+θeλb​

代入θ=X−ab−a，并利用1−θ=b−Xb−a，可得：eλX≤b−Xb−aeλa+X−ab−aeλb\begin{array}{cc} 代入 \theta = \frac{X - a}{b - a} ，并利用 1-\theta = \frac{b - X}{b - a} ，可得：\\ e^{\lambda X} \leq \frac{b - X}{b-a} e^{\lambda a} + \frac{X - a}{b-a} e^{\lambda b} \end{array} 代入θ=b−aX−a​，并利用1−θ=b−ab−X​，可得：eλX≤b−ab−X​eλa+b−aX−a​eλb​

对上式两边取期望，并注意到X的期望为0：

E[eλX]≤b−E[X]b−aeλa+E[X]−ab−aeλb=bb−aeλa−ab−aeλb\begin{aligned} \mathbb{E}[e^{\lambda X}] &\leq \frac{b - \mathbb{E}[X]}{b-a} e^{\lambda a} + \frac{\mathbb{E}[X] - a}{b-a} e^{\lambda b} \\ &= \frac{b}{b-a} e^{\lambda a} - \frac{a}{b-a} e^{\lambda b} \end{aligned} E[eλX]​≤b−ab−E[X]​eλa+b−aE[X]−a​eλb=b−ab​eλa−b−aa​eλb​

这个地方其实可以敏锐地发现这其实是两点分布的期望（然而一上来直接用两点分布放缩并不严格！！！）

引入两点分布的概率p，我们可以将上式写为：

E[eλX]≤(1−p)eλa+peλb=L(λ)\begin{array}{cc} \mathbb{E}[e^{\lambda X}] \leq (1-p)e^{\lambda a} + p e^{\lambda b} = L(\lambda) \end{array} E[eλX]≤(1−p)eλa+peλb=L(λ)​

下面我们进一步考察其上界，我们致力于证明：

L(λ)=(1−p)eλa+peλb≤exp⁡(λ2(b−a)28)\begin{array}{cc} L(\lambda) = (1-p)e^{\lambda a} + p e^{\lambda b} \leq \exp\left(\frac{\lambda^2 (b-a)^2}{8}\right) \end{array} L(λ)=(1−p)eλa+peλb≤exp(8λ2(b−a)2​)​

以区间长度作为变量，令u=λ(b−a)，且令ϕ=p∈[0,1]。则：a=−ϕ(b−a),b=(1−ϕ)(b−a)代入L(λ)得：L(λ)=(1−ϕ)e−λϕ(b−a)+ϕeλ(1−ϕ)(b−a)即：L(λ)=(1−ϕ)e−ϕu+ϕe(1−ϕ)u考虑其对数g(u)=ln⁡L(λ)=ln⁡[(1−ϕ)e−ϕu+ϕe(1−ϕ)u]\begin{array}{cc} 以区间长度作为变量，令 u = \lambda (b-a) ，且令 \phi = p \in [0,1] 。则：\\ a = -\phi (b-a), \quad b = (1-\phi)(b-a) 代入 L(\lambda) 得：\\ L(\lambda) = (1-\phi)e^{-\lambda \phi (b-a)} + \phi e^{\lambda (1-\phi)(b-a)}\\ 即： L(\lambda) = (1-\phi)e^{-\phi u} + \phi e^{(1-\phi)u}\\ 考虑其对数 g(u) = \ln L(\lambda) = \ln\left[ (1-\phi)e^{-\phi u} + \phi e^{(1-\phi)u} \right] \end{array} 以区间长度作为变量，令u=λ(b−a)，且令ϕ=p∈[0,1]。则：a=−ϕ(b−a),b=(1−ϕ)(b−a)代入L(λ)得：L(λ)=(1−ϕ)e−λϕ(b−a)+ϕeλ(1−ϕ)(b−a)即：L(λ)=(1−ϕ)e−ϕu+ϕe(1−ϕ)u考虑其对数g(u)=lnL(λ)=ln[(1−ϕ)e−ϕu+ϕe(1−ϕ)u]​

现在，我们再对g(u)进行暴力的Taylor展开：

显然，上述函数在零处为零，我们计算0处的一、二阶导数：

g′(u)=−(1−ϕ)ϕe−ϕu+ϕ(1−ϕ)e(1−ϕ)u(1−ϕ)e−ϕu+ϕe(1−ϕ)u\begin{array}{cc} g'(u) = \frac{-(1-\phi)\phi e^{-\phi u} + \phi(1-\phi) e^{(1-\phi)u}}{(1-\phi)e^{-\phi u} + \phi e^{(1-\phi)u}} \end{array} g′(u)=(1−ϕ)e−ϕu+ϕe(1−ϕ)u−(1−ϕ)ϕe−ϕu+ϕ(1−ϕ)e(1−ϕ)u​​

代入u=0，容易算出g在0处的一阶导为零

g′′(u)=ϕ(1−ϕ)[(1−ϕ)e−ϕu+ϕe(1−ϕ)u]2−[−(1−ϕ)ϕe−ϕu+ϕ(1−ϕ)e(1−ϕ)u]2[(1−ϕ)e−ϕu+ϕe(1−ϕ)u]2则有：g′′(0)=ϕ(1−ϕ)≤14\begin{array}{cc} g''(u) = \frac{\phi(1-\phi)\left[ (1-\phi)e^{-\phi u} + \phi e^{(1-\phi)u} \right]^2 - \left[ -(1-\phi)\phi e^{-\phi u} + \phi(1-\phi) e^{(1-\phi)u} \right]^2}{\left[ (1-\phi)e^{-\phi u} + \phi e^{(1-\phi)u} \right]^2}\\ 则有： g''(0) =\phi(1-\phi) \leq \frac{1}{4} \end{array} g′′(u)=[(1−ϕ)e−ϕu+ϕe(1−ϕ)u]2ϕ(1−ϕ)[(1−ϕ)e−ϕu+ϕe(1−ϕ)u]2−[−(1−ϕ)ϕe−ϕu+ϕ(1−ϕ)e(1−ϕ)u]2​则有：g′′(0)=ϕ(1−ϕ)≤41​​

则由Taylor展开，有：

g(u)≤12⋅14u2=u28\begin{array}{cc} g(u) \leq \frac{1}{2} \cdot \frac{1}{4} u^2 = \frac{u^2}{8} \end{array} g(u)≤21​⋅41​u2=8u2​​

因此，将u的值带入后有：

L(λ)≤exp⁡(λ2(b−a)28)\begin{array}{cc} L(\lambda) \leq \exp\left( \frac{\lambda^2 (b-a)^2}{8} \right) \end{array} L(λ)≤exp(8λ2(b−a)2​)​

引理得证，现在可以开始正式证明霍夫丁不等式：

对Xi∈[ai,bi]，对任意ε>0，有：P(X‾−μ≥ε)≤exp⁡[−2n2ε2∑i=1n(bi−ai)2]\begin{array}{cc} 对 X_i \in [a_i, b_i] ，对任意 \varepsilon > 0 ，\\有： P(\overline{X} - \mu \geq \varepsilon) \leq \exp\left[ -\frac{2n^2\varepsilon^2}{\sum_{i=1}^n (b_i - a_i)^2} \right] \end{array} 对Xi​∈[ai​,bi​]，对任意ε>0，有：P(X−μ≥ε)≤exp[−∑i=1n​(bi​−ai​)22n2ε2​]​

首先构造：Z(λ)=exp⁡[λn(X‾−μ)]那么P(X‾−μ≥ε)等价于P(Z>eλnε)，显然Z(λ)>0\begin{array}{cc} 首先构造： Z(\lambda) = \exp\left[ \lambda n (\overline{X} - \mu) \right] \\ 那么 P(\overline{X} - \mu \geq \varepsilon) 等价于 P(Z > e^{\lambda n \varepsilon}) ，显然 Z(\lambda) > 0 \end{array} 首先构造：Z(λ)=exp[λn(X−μ)]那么P(X−μ≥ε)等价于P(Z>eλnε)，显然Z(λ)>0​

由Markov定理：P(Z>eλnε)≤E(Z)eλnε由 Markov 定理： P(Z > e^{\lambda n \varepsilon}) \leq \frac{E(Z)}{e^{\lambda n \varepsilon}} 由Markov定理：P(Z>eλnε)≤eλnεE(Z)​

现在我们考虑将Z(λ)展开：Z(λ)=exp⁡[λn(1n∑i=1nXi−μ)]=exp⁡[λ∑i=1n(Xi−μ)]=∏i=1nexp⁡[λ(Xi−μ)]\begin{array}{cc} 现在我们考虑将 Z(\lambda) 展开：\\ Z(\lambda) = \exp\left[ \lambda n \left( \frac{1}{n}\sum_{i=1}^n X_i - \mu \right) \right]\\ = \exp\left[ \lambda \sum_{i=1}^n (X_i - \mu) \right] = \prod_{i=1}^n \exp\left[ \lambda (X_i - \mu) \right] \end{array} 现在我们考虑将Z(λ)展开：Z(λ)=exp[λn(n1​∑i=1n​Xi​−μ)]=exp[λ∑i=1n​(Xi​−μ)]=∏i=1n​exp[λ(Xi​−μ)]​

令Yi=Xi−μ，则有：Yi∈[ai−μ,bi−μ]，且E(Yi)=0\begin{array}{cc} 令 Y_i = X_i - \mu ，则有： Y_i \in [a_i - \mu, b_i - \mu] ，且 E(Y_i) = 0 \end{array} 令Yi​=Xi​−μ，则有：Yi​∈[ai​−μ,bi​−μ]，且E(Yi​)=0​

由Hoeffding引理：

∏i=1nE(eλYi)≤exp⁡(λ2∑i=1n(bi−ai)28)记∑i=1n(bi−ai)2=t2，则：E(Z)eλnε≤exp⁡[λ2t28−λnε]\begin{array}{cc} \prod_{i=1}^n E(e^{\lambda Y_i}) \leq \exp\left( \frac{\lambda^2 \sum_{i=1}^n (b_i - a_i)^2}{8} \right) \\记 \sum_{i=1}^n (b_i - a_i)^2 = t^2 ，则： \frac{E(Z)}{e^{\lambda n \varepsilon}} \leq \exp\left[ \frac{\lambda^2 t^2}{8} - \lambda n \varepsilon \right] \end{array} ∏i=1n​E(eλYi​)≤exp(8λ2∑i=1n​(bi​−ai​)2​)记∑i=1n​(bi​−ai​)2=t2，则：eλnεE(Z)​≤exp[8λ2t2​−λnε]​

欲使这个不等式恒成立，我们需要对于任意的指数均成立，则我们只需要考虑这个二次函数的最小值，而这个结果是显然的：

P(X‾−μ≥ε)≤exp⁡(−2n2ε2t2),取等时，λ=4nεt2P(\overline{X} - \mu \geq \varepsilon) \leq \exp\left( -\frac{2n^2 \varepsilon^2}{t^2} \right),取等时，\lambda=\frac{4n\varepsilon}{t^2} P(X−μ≥ε)≤exp(−t22n2ε2​),取等时，λ=t24nε​

至此，Horffding不等式得证！

3.联合界的证明：

P(⋃i=1nXi)≤∑i=1nP(Xi) P\left( \bigcup_{i=1}^n X_i \right) \leq \sum_{i=1}^n P(X_i) P(i=1⋃n​Xi​)≤i=1∑n​P(Xi​)

采用数学归纳法：

当n=2时：P(X1∪X2)=P(X1)+P(X2)−P(X1∩X2)，因P(X1∩X2)≥0，故P(X1∪X2)≤P(X1)+P(X2)，不等式成立。假设对于n−1，不等式成立：即P(⋃i=1n−1Xi)≤∑i=1n−1P(Xi)。下面证明对于n，不等式也成立：令Q=⋃i=1n−1Xi（将前n−1个事件的并视为一个新事件），则：P(⋃i=1nXi)=P(Q∪Xn)由n=2时的结论，有：P(Q∪Xn)≤P(Q)+P(Xn)结合归纳假设P(Q)≤∑i=1n−1P(Xi)，代入得：P(Q∪Xn)≤∑i=1n−1P(Xi)+P(Xn)=∑i=1nP(Xi)。故P(⋃i=1nXi)≤∑i=1nP(Xi)，归纳成立。\begin{array}{cc} 当 n=2 时：\\ P(X_1 \cup X_2) = P(X_1) + P(X_2) - P(X_1 \cap X_2) ，\\ 因 P(X_1 \cap X_2) \geq 0 ，\\故 P(X_1 \cup X_2) \leq P(X_1) + P(X_2) ，不等式成立。\\ 假设对于 n-1 ，不等式成立：\\ 即 P\left( \bigcup_{i=1}^{n-1} X_i \right) \leq \sum_{i=1}^{n-1} P(X_i) 。\\ 下面证明对于 n ，不等式也成立：\\ 令 Q = \bigcup_{i=1}^{n-1} X_i （将前 n-1 个事件的并视为一个新事件），则：\\ P\left( \bigcup_{i=1}^n X_i \right) = P(Q \cup X_n) \\由 n=2 时的结论，有：\\ P(Q \cup X_n) \leq P(Q) + P(X_n) \\结合归纳假设 P(Q) \leq \sum_{i=1}^{n-1} P(X_i) ，代入得：\\ P(Q \cup X_n) \leq \sum_{i=1}^{n-1} P(X_i) + P(X_n) = \sum_{i=1}^n P(X_i) 。\\ 故 P\left( \bigcup_{i=1}^n X_i \right) \leq \sum_{i=1}^n P(X_i) ，归纳成立。 \end{array} 当n=2时：P(X1​∪X2​)=P(X1​)+P(X2​)−P(X1​∩X2​)，因P(X1​∩X2​)≥0，故P(X1​∪X2​)≤P(X1​)+P(X2​)，不等式成立。假设对于n−1，不等式成立：即P(⋃i=1n−1​Xi​)≤∑i=1n−1​P(Xi​)。下面证明对于n，不等式也成立：令Q=⋃i=1n−1​Xi​（将前n−1个事件的并视为一个新事件），则：P(⋃i=1n​Xi​)=P(Q∪Xn​)由n=2时的结论，有：P(Q∪Xn​)≤P(Q)+P(Xn​)结合归纳假设P(Q)≤∑i=1n−1​P(Xi​)，代入得：P(Q∪Xn​)≤∑i=1n−1​P(Xi​)+P(Xn​)=∑i=1n​P(Xi​)。故P(⋃i=1n​Xi​)≤∑i=1n​P(Xi​)，归纳成立。​

PRAT2泛化误差上界的求解

经过上述繁琐的证明，我们终于完成了一切准备工作，现在开始正式的证明：

对单个假设R^(f)（测试误差）和R(f)（泛化误差），泛化误差是测试误差的期望，则由Hoeffding不等式可知：P(R(f)−R^(f)≥ε)≤Cexp⁡(−2nε2)\begin{array}{cc} 对单个假设 \hat{R}(f) （测试误差）和 R(f) （泛化误差），泛化误差是测试误差的期望，\\则由Hoeffding不等式可知：\\ P\left( R(f) - \hat{R}(f) \geq \varepsilon \right) \leq C\exp\left( -{2n\varepsilon^2} \right) \end{array} 对单个假设R^(f)（测试误差）和R(f)（泛化误差），泛化误差是测试误差的期望，则由Hoeffding不等式可知：P(R(f)−R^(f)≥ε)≤Cexp(−2nε2)​

这里的C是一个常数，与误差所取的具体函数形式有关（比如说，0-1损失中C=1）

设假设空间为F={f1,f2,...,fd}（含d个假设），我们需要控制,至少存在一个假设f∈F，使得R(f)−R^(f)≥ε的概率。由联合界，该事件的概率满足：P(∃f∈F, R(f)−R^(f)≥ε)=P(⋃f∈F{R(f)−R^(f)≥ε})≤∑f∈FP(R(f)−R^(f)≥ε)\begin{array}{cc} 设假设空间为 \mathcal{F} = \{f_1, f_2, ..., f_d\} （含 d 个假设），\\我们需要控制,至少存在一个假设 f \in \mathcal{F} ，使得 R(f) - \hat{R}(f) \geq \varepsilon 的概率。\\ 由联合界，该事件的概率满足：\\ P\left( \exists f \in \mathcal{F},\ R(f) - \hat{R}(f) \geq \varepsilon \right) = P\left( \bigcup_{f \in \mathcal{F}} \left\{ R(f) - \hat{R}(f) \geq \varepsilon \right\} \right) \\\leq \sum_{f \in \mathcal{F}} P\left( R(f) - \hat{R}(f) \geq \varepsilon \right) \end{array} 设假设空间为F={f1​,f2​,...,fd​}（含d个假设），我们需要控制,至少存在一个假设f∈F，使得R(f)−R^(f)≥ε的概率。由联合界，该事件的概率满足：P(∃f∈F, R(f)−R^(f)≥ε)=P(⋃f∈F​{R(f)−R^(f)≥ε})≤∑f∈F​P(R(f)−R^(f)≥ε)​

将单假设的Hoeffding上界代入，得：P(∃f∈F, R(f)−R^(f)≥ε)≤d⋅Cexp⁡(−2nε2)\begin{array}{cc} 将单假设的Hoeffding上界代入，得：\\ P\left( \exists f \in \mathcal{F},\ R(f) - \hat{R}(f) \geq \varepsilon \right) \leq d \cdot C\exp\left( -2n\varepsilon^2 \right) \end{array} 将单假设的Hoeffding上界代入，得：P(∃f∈F, R(f)−R^(f)≥ε)≤d⋅Cexp(−2nε2)​

记δ=d⋅exp⁡(−2nε2)（控制坏事件的概率），解出ε：ε=ln⁡(d/δ)2n因此，以至少1−δ的概率，对所有f∈F成立：R(f)≤R^(f)+Cln⁡(d/δ)2n\begin{array}{cc} 记 \delta = d \cdot \exp\left( -2n\varepsilon^2 \right) （控制坏事件的概率），解出 \varepsilon ：\\ \varepsilon = \sqrt{\frac{\ln(d/\delta)}{2n}} \\ 因此，以至少 1-\delta 的概率，对所有 f \in \mathcal{F} 成立：\\ R(f) \leq \hat{R}(f) + C\sqrt{\frac{\ln(d/\delta)}{2n}} \end{array} 记δ=d⋅exp(−2nε2)（控制坏事件的概率），解出ε：ε=2nln(d/δ)​​因此，以至少1−δ的概率，对所有f∈F成立：R(f)≤R^(f)+C2nln(d/δ)​​​

至此，我们完成了泛化误差上界的求解（其实就是Hoeffding那块从0开始证明比较复杂）

严格而言，d应该指的是假设空间的VC维，期待后续的详细讨论。
